### Step 1: Update Domain Name (CRITICAL!)

- Line 21: Twitter Card URL
- Line 40: Google Verification (leave empty for now)

### Step 2: Update Sitemap

Open `sitemap.xml` and replace all `https://myprofile.dev` with your actual domain

### Step 3: Update Robots

Open `robots.txt` and replace `https://myprofile.dev` with your actual domain

### Step 4: Rename and Deploy

```bash
# Rename to main file
mv index-new.html index.html

# Upload to your server:
# - index.html
# - robots.txt
# - sitemap.xml
# - .htaccess (if Apache server)
# - css/ folder
# - js/ folder
# - assets/ folder
```

### Step 5: Verify with Google

1. Go to [Google Search Console](https://search.google.com/search-console)
2. Add your website property
3. Choose HTML tag verification
4. Copy the code and add to `index-new.html` line 40
5. Upload updated file
6. Click Verify in GSC

### Step 6: Submit Sitemap

1. In Google Search Console
2. Go to Sitemaps section
3. Enter: `https://yourdomain.com/sitemap.xml`
4. Submit

### Step 7: Monitor

1. Wait 2-4 weeks for indexing
2. Check Google Search Console for impressions
3. Monitor rankings for "Rizbi Islam"
